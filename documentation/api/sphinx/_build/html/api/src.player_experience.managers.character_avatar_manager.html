src.player_experience.managers.character_avatar_manager module

Character creation, customization, and lifecycle management.

This module provides the CharacterAvatarManager service that handles character CRUD operations, enforces character limits, and integrates with the existing CharacterDevelopmentSystem for therapeutic profiles.
