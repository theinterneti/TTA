src.agent_orchestration.messaging module

Message passing data structures for Agent Orchestration (Task 2.2, 4.2).

Adds reliability primitives used by MessageCoordinator implementations: - FailureType to distinguish transient vs permanent failures - ReceivedMessage reservation wrapper for ack/nack with visibility timeout - QueueMessage extended with delivery_attempts and timestamps
