src.api_gateway.core.circuit_breaker module

Circuit breaker pattern implementation for the API Gateway.

This module provides circuit breaker functionality to protect backend services from cascading failures and provide graceful degradation.

Circuit breaker implementation for protecting backend services.

The circuit breaker monitors service health and automatically opens when failure thresholds are exceeded, providing fail-fast behavior and automatic recovery testing.
