src.agent_orchestration.workflow_manager module

WorkflowManager core functionality (Task 3.1, extended for 3.2 integration points).

Provides registration, validation, and a minimal execution entrypoint for workflow definitions. Adds optional LangGraph integration: build_graph and execute_graph delegate to LangGraph when available via LangGraphWorkflowBuilder and LangGraphExecutor; otherwise they return stub responses.
