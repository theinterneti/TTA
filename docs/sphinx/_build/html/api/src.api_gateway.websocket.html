src.api_gateway.websocket package

WebSocket proxying and real-time communication handling for the API Gateway.

This module provides WebSocket proxy functionality for real-time therapeutic sessions, chat services, and narrative interactions with comprehensive connection management, message routing, and therapeutic safety features.
