src.player_experience.api.routers.players module

Player management router for the Player Experience API.

This module provides REST endpoints for player profile CRUD operations with authentication, authorization, and comprehensive API documentation.

Delete a player profile.

Permanently deletes the player profile and all associated data. This operation implements the GDPR right to erasure and cannot be undone. Only the profile owner can delete their own profile.

  • current_player (TokenData) – Current authenticated player (from JWT token)

  • manager (PlayerProfileManager) – Player profile manager dependency
