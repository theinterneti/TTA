class src.components.narrative_arc_orchestrator_component.NarrativeArcOrchestratorComponent(config)[source]

Bases: Component

Main component for narrative arc orchestration.

This component manages storytelling across multiple temporal scales while maintaining therapeutic integration and player agency. It coordinates with Neo4j for narrative state storage, Redis for session management, and the Interactive Narrative Engine for scene-level interactions.
