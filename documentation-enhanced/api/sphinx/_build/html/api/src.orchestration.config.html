TTA Configuration

This module provides configuration management for the TTA project.

Classes:

TTAConfig: Configuration manager for the TTA project

Example

```python from src.orchestration.config import TTAConfig

# Create a configuration object config = TTAConfig()

# Get a configuration value value = config.get(“tta.dev.enabled”)

# Set a configuration value config.set(“tta.dev.enabled”, True)

# Save the configuration config.save() ```
