src.player_experience.database.user_auth_schema module

User Authentication Schema Manager for Neo4j.

This module manages the Neo4j schema for user authentication data, including User nodes, constraints, indexes, and relationships with PlayerProfile nodes for the therapeutic text adventure platform.
