# TTA Therapeutic Platform - Component Library

Welcome to the **TTA (Therapeutic Text Adventure) Platform** component library! This Storybook provides interactive documentation and testing capabilities for all components in our therapeutic platform.

## üéØ Purpose

The TTA platform is designed to provide personalized therapeutic experiences through interactive text adventures. This component library serves as:

- **Visual Documentation**: See how components look and behave in different states
- **Interactive Testing**: Test component functionality without backend dependencies
- **Development Tool**: Develop and refine components in isolation
- **Design System**: Maintain consistency across the therapeutic platform

## üß© Featured Components

### TherapeuticGoalsSelector
A comprehensive component for therapeutic intake and preference setting, helping users identify their primary areas of focus for personalized therapeutic experiences.

**Key Features:**
- Dual tab interface for Goals and Primary Concerns
- 15+ predefined therapeutic goals across 5 categories
- Custom goal and concern entry
- Quick selection presets
- Real-time selection summary
- Full accessibility support
